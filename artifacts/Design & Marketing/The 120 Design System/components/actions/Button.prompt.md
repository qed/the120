The CTA button — mono uppercase label, 10px radius. Red `primary` is the single loud action per view ("Join the 120"); pair it with `ghost` ("Book a call").

```jsx
<Button variant="primary" href="/join">Join the 120</Button>
<Button variant="ghost" href="/book">Book a call</Button>
```

On dark/blue/red surfaces use `white` (filled) + `ghostLight` (bordered). Use `ink` for a secondary solid on light.
